Proxy¶

Proxy para acceder a la API REST de Satellite Images.

Este módulo tiene funciones específicas para obtener imágenes por satélite de los distintos servicios implementados por la API REST Satellite Images de forma homogénea.

Returns

Lista de servicios disponibles en el sistema

dict(str->list[str])

Proxy.getMetadata(point, **kwargs)[source]¶

Función para obtener metadatos de imágenes que coincidan con los parámetros

Esta clase llama a la ruta /getMetadadata paraobtener los metadatos de la imágenes coincidentes con los parámetros dados en la consulta.

Parameters

  • point (str) –

    • Descripción: Punto perteneciente a la imagen en latitud y longitud.

    • Formato: “latitud,longitud”

    • Ejemplo: “41.800507792693544,-2.304528141507549”

  • **kwargs (dict[str->str]) –

    Lista de parámetros que haya introducido el usuario. Estos pueden incluir:

    • servicio

      • Descripción: Servicio o servicios de los que se quieren obtener metadatos.

    • size

      • Descripción: Tamaño en píxeles de la imagen.

      • Formato: “altura,anchura”

      • Ejemplo: “1000,1000”

    • format

      • Descripción: Formato de la imagen.

    • resolution

      • Descripción: Resolución máxima en metros/pixel de la imagen que queremos obtener.

      • Ejemplo: “3.5”

    • beforeDate

      • Descripción: Obtener únicamente imágenes anteriores a la fecha dada.

      • Formato: “dd-mm-aaaa”

      • Ejemplo: “12-09-2005”

    • afterDate

      • Descripción: Obtener únicamente imágenes posteriores a la fecha dada.

      • Formato: “dd-mm-aaaa”

      • Ejemplo: “12-09-2005”

    • maxResults

      • Descripción: Resolución máxima en metros/pixel de la imagen que queremos obtener.

      • Ejemplo: “3.5”

    • itemTypes

      • Descripción: Tipos de imágenes que se quiere que se devuelvan (Para Planet Labs)

      • Ejemplo: “PSScene4Band,PSOrthoTile”

Returns

Metadatos en formato GeoJSON o un mensaje de error si se hubiera producido alguno.

Return type

str

Proxy.getImage(provider, **kwargs)[source]¶

Función para obtener una imagen

Dados unos parámetros, se devuelve la imagen coincidente con estos además de sus metadatos correspondientes. En algunos servicios el parámetro point será obligatorio mientras que en otros lo séra el parámetro id, dependiendo de las características de este servicio.

Parameters

  • provider (str) –

    Servicio del que se quiere obtener la imagen

    • Descripción: Proveedor o servicio del que se quiere obtener la imagen

  • **kwargs (dict[str->str]) –

    Lista de parámetros que haya introducido el usuario. Estos pueden incluir:

    • point

      • Descripción: Punto perteneciente a la imagen en latitud y longitud.

      • Formato: “latitud,longitud”

      • Ejemplo: “41.800507792693544,-2.304528141507549”

    • size

      • Descripción: Tamaño en píxeles de la imagen.

      • Formato: “altura,anchura”

      • Ejemplo: “1000,1000”

    • id

      • Descripción: Id de la imagen que se quiere obtener

Returns

Imagen en bytes y sus metadatos

Return type

Tuple(bytes, str)

Returns

Mensaje de error si se ha producido alguno cuando se intentaba obtener la imagen

Return type

Tuple(str, None)
