ApiRest¶

Aplicación REST para la descarga de imágenes de distintos servicios

Esta aplicación permite descargar imágenes por satélite de servicios como Google Maps o Bing Maps. Los módulos implementados son independientes y es posible añadir más de forma transparente.

Example

Para ejecutar este script se debe ejecutar en una terminal en la carpeta donde se almacene este archivo el siguiente código:

$ python ApiRest.py

Tras esto se podrán realizar llamadas a la API desde el servidor:

http://localhost:5000/

class ApiRest.Image[source]¶

Bases: flask_restful.Resource

Clase para obtener imágenes por satélite y sus metadatos de los servicios

Esta clase llama a la función getImage para obtener una imagen del servicio especificado con los parámetros que se le hayan pasado en la consulta. Se puede acceder mediante la ruta /getImage.

Parameters

args (dict[str->str]) –

Este parámetro se obtendrá de la consulta que se haga. Cada uno de los parámetros tendrán funciones generales o específicas para uno o más de los servicios. Se pueden insertar los siguientes parámetros:

  • Obligatorio:
    • servicio

      • Descripción: Servicio del que se quiere obtener la imagen.

  • Obligatorio uno de ellos:
    • id

      • Descripción: Id de la imagen

    • point

      • Descripción: Punto perteneciente a la imagen en latitud y longitud.

      • Formato: “latitud,longitud”

      • Ejemplo: “41.800507792693544,-2.304528141507549”

  • Opcionales:
    • size

      • Descripción: Tamaño en píxeles de la imagen.

      • Formato: “altura,anchura”

      • Ejemplo: “1000,1000”

    • format

      • Descripción: Formato de la imagen.

    • resolution

      • Descripción: Resolución máxima en metros/pixel de la imagen que queremos obtener.

      • Ejemplo: “3.5”

    • beforeDate

      • Descripción: Obtener únicamente imágenes anteriores a la fecha dada.

      • Formato: “dd-mm-aaaa”

      • Ejemplo: “12-09-2005”

    • afterDate

      • Descripción: Obtener únicamente imágenes posteriores a la fecha dada.

      • Formato: “dd-mm-aaaa”

      • Ejemplo: “12-09-2005”

    • maxResults

      • Descripción: Número máximo de resultados que queremos obtener por servicio

      • Ejemplo: “5”

    • itemTypes

      • Descripción: Tipos de imágenes que se quiere que se devuelvan

      • Ejemplo: “PSScene4Band,PSOrthoTile”

Returns

Respuesta con la imagen solicitada en bytes con los metadatos en la cabecera y el código HTTP 200.

Return type

Response

Returns

Respuesta con el estado de una imagen que se ha solicitado descargar pero necesita tiempo su activación y el código HTTP 202.

Return type

Response

Returns

Respuesta con un mensaje de error si alguno ha ocurrido con el código HTTP 400.

Return type

Response
